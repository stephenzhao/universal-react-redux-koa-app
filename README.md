# universal-react-redux-koa-app
一个居于koa + react + antd 的同构后台项目;


## 初步设想的技术栈
* ES6
* Babel
* [React](https://facebook.github.io/react/)
* [ReactRouter](https://github.com/rackt/react-router)
* [koajs](http://koajs.org)
* [webpack](https://webpack.github.io/)
* antd
